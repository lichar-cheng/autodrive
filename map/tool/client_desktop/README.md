# Desktop Client

`client_desktop` is the native Tk desktop client for AutoDrive map tooling.

## Current Coverage

The desktop client now tracks the browser client feature set closely:

- WebSocket stream connection with reconnect, ping/pong, checksum/seq/lag diagnostics, and `/health` polling
- Scan accumulation and map canvas rendering
- Local `.slam` map save and load
- Map exports for `PGM`, `YAML`, and `JSON`
- Second-stage map editing
- Noise erase and obstacle-line drawing
- POI add, batch POI add, delete, geo apply, and clipboard copy
- Path connect by POI name
- Path connect by free-point clicks
- Obstacle-aware path planning with safety clearance
- Auto loop generation and closed-loop validation
- Move controls with repeated keyboard driving and stop-on-keyup
- Buffered camera snapshot refresh
- Panel visibility toggles

## Run

```bash
cd client_desktop
python3 -m pip install -r requirements.txt
python3 run_client.py
```

On Windows, replace `python3` with `python` if needed.

## Package

### Windows

```bat
cd client_desktop
build_windows.bat
```

Default output is generated by PyInstaller in `dist/maptool_001_20260408.exe`.
You can override the app name with `APP_NAME`, for example:

```bat
set APP_NAME=maptool_001_20260409
build_windows.bat
```

### Linux

```bash
cd client_desktop
chmod +x build_linux.sh
./build_linux.sh
```

Default output is generated by PyInstaller in `dist/maptool_001_20260408`.
You can override the app name with `APP_NAME`, for example:

```bash
APP_NAME=maptool_001_20260409 ./build_linux.sh
```

### macOS

```bash
cd client_desktop
chmod +x build_macos.sh
./build_macos.sh
```

Default output is generated by PyInstaller in `dist/maptool_001_20260408.app`.
You can override the app name with `APP_NAME`, for example:

```bash
APP_NAME=maptool_001_20260409 ./build_macos.sh
```

This script must be run on macOS.
